## Описание проекта

MyCloud — это облачное хранилище файлов с двухуровневой системой прав доступа (администратор/пользователь). Приложение позволяет:

- Регистрироваться и аутентифицироваться

- Загружать, скачивать, переименовывать и удалять файлы

- Добавлять комментарии к файлам

- Создавать специальные ссылки для публичного доступа к файлам

- Администраторам управлять пользователями и их хранилищами
